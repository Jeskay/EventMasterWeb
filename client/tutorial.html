You can find invite link right on the navigation bar from our website or click "Add to Server" button in the bot's profile. Make sure you give the bot all necessary permissions to create and modify channels as well as to create slash commands. Otherwise, you won't be able to use it.

Setting the bot uparrow_drop_up
Setting the bot upclose

First of all players need a room to play. Use command /owner setup to select category where bot will create voice and text channels for players and voice channel where people will join to initiate an event. That's minimum setting you can start with. However, settings are not limited to the command above.
/owner limit_events - to set max amount of events allowed for creation at the same time.
/owner limit_votes - to set minimum amount of votes for single user to be elected as a host.
/owner event_role - to set role which will be notified when new event is hosted at the server.
/owner logging - to set channel where bot will information about actions related to events.
/owner notifications - to set channel where bot will publish notifications about events.
/owner set_admin - to give a user permissions to block and unblock users from joining events.
/owner remove_admin - to remove the privileges described in previous command.
Electing the hostarrow_drop_up
Electing the hostclose

When there are enough people, the election will start. Participants can vote for another person to become the host and manange te event. There are two options for voting - in EventMaster's DM using /player voteid or in server's chat using /member vote . Bot will send a message to the event's chat when election is over.

Managing eventarrow_drop_up
Managing eventclose

Newly elected host usually performs the following steps: /host announce - to publish announcement about the event so more people will join the event. /host start - to start the event and post this information in log channel. /host finishgame - to end current event, record results and delete the channel Whenever someone violates server's policy or annoy other users owner and admins can use the bot to block such users from joining events: /admin block - add user to the blacklist and block him from joining events. /admin unblock - remove user from the blacklist and allow him to join events. /admin blacklist - show all users blocked from joining events on this server.

Profilesarrow_drop_up
Profilesclose

Every user who participated at least one event has guild profile and global profile data. /member guildprofile - Shows user profile at the specific server. /player profile - Shows global profile of the user. /system rating - Global rating of the most active users. /member guildprofile - Rating of the most active users on the server.

Subscriptionsarrow_drop_up
Subsciptionsclose

To receive notifications about specific events a user must subscribe to it. Use /system subscribe command to be notified about the event. For example, /system subscribe title:chess will notify user when public announcement includes "#chess" string in it. /system unsubscribe - to remove subscription from your list. /system subscriptions - shows list of your subscriptions.
